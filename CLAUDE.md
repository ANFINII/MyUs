# Claude Code 開発ルール

## コーディングルール確認（必須）

コードを書いた後、必ず対象ディレクトリのルールファイルを確認し、ルールに適合しているか検証すること。

### バックエンド（myus/）
- ルールファイル: `myus/CLAUDE.md`
- 対象: Python/Djangoコード
- 確認後: `mypy` でリンターチェックを実行

### フロントエンド（frontend/）
- ルールファイル: `frontend/CLAUDE.md`
- 対象: TypeScript/React/Next.jsコード
- 確認後: `npm run lint` でリンターチェックを実行

## 確認フロー

1. コードを書く
2. 対象ディレクトリのCLAUDE.mdを読む
3. ルールに適合しているか確認し、違反があれば修正
4. リンターチェックを実行してエラーがあれば修正
